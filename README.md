# Tool

Una raccolta unificata di guide essenziali e riferimenti rapidi per strumenti Linux e Docker, pensata per l'uso quotidiano in ambito DevOps e sistemistica.

---

## Contenuto

### [linux/](./linux/)
Guida completa ai tool Linux più utilizzati (Debian, Ubuntu e derivate), organizzati per categoria:
Networking, System Monitoring, Package Management, File Management, Process Management, Disk & Storage, System Information, Security, User Management, Task Scheduling, Shell Utilities.

- [`linux/tools.md`](./linux/tools.md) — Riferimento completo con comandi, esempi e distribuzioni target.

### [docker/](./docker/)
Riferimento pratico ai comandi Docker e Docker Compose, con esempi reali per lo sviluppo e la produzione:
Immagini, Container, Volumi, Reti, Compose, Debug e Ispezione.

- [`docker/tools.md`](./docker/tools.md) — Riferimento completo con comandi, esempi e flussi di diagnosi.
