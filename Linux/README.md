# Linux Tools

Una guida essenziale e strutturata ai tool più utilizzati nei sistemi Linux, con particolare attenzione a Debian, Ubuntu e derivate.

## Contenuto

Il repository contiene:

- **`tools.md`** — Elenco completo di tool Linux organizzati per categoria:
  - Networking
  - System Monitoring & Performance
  - Package Management
  - File Management & Text Processing
  - Process Management
  - Disk & Storage
  - System Information
  - Security
  - User & Group Management
  - Task Scheduling & Automation
  - Shell & Terminal Utilities

Ogni tool include: nome, descrizione, comando principale, esempio d'uso e distribuzioni target.

## Struttura del file

```markdown
### `nome-tool`
**Descrizione breve.**

- **Comando:** `comando principale`
- **Esempio:** `esempio d'uso`
- **Distribuzioni:** Elenco distribuzioni
```

## Requisiti

Nessuno. I tool elencati sono disponibili nativamente o tramite i gestori pacchetti delle rispettive distribuzioni.
